# Global Country Intelligence Map

An interactive country intelligence app built with HTML, CSS, and JavaScript.  
It helps users explore countries visually, compare real public data, and make more informed decisions around relocation, safety, economic conditions, jobs, and long-term planning.

## What This App Does

This project turns public country-level datasets into a more practical and visual decision tool.

Users can:

- hover and click countries on an interactive world map
- explore real country statistics from public sources
- compare countries across economic, safety, and relocation indicators
- switch between visual map layers like economy, family safety, hazards, jobs, and decision fit
- use decision modes such as family move, career growth, retirement, study abroad, remote work, and investment
- review country story summaries written from real indicators
- inspect crime trends, GDP trends, natural hazard exposure, weather, alerts, and other country details
- export data and share snapshots

## Data Sources

This app uses open public data sources, including:

- REST Countries
- World Bank API
- WHO
- UN SDG API
- ND-GAIN
- EM-DAT
- OpenFlights
- Open-Meteo

Note: some views in the app use composite scoring models built from real public indicators. These are designed to help users interpret data more easily and should not be treated as official advice.
