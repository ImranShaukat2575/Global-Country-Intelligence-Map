Sources:

  • REST Countries (country metadata)
  • World Bank Indicators API (GDP, income, unemployment, poverty, Gini, tourism, military)
  • IMF DataMapper API (inflation, debt, current account)
  • UN SDG API (crime rate trend, SDG 16.1.1)
  • WHO GHO API (health indicators snapshot)
  • ND-GAIN (climate vulnerability/readiness snapshot)
  • EM-DAT via HDX (disaster profile snapshot)
  • UN Tourism (UNWTO) (tourism methodology/source, surfaced through World Bank tourism indicators)
  • OpenFlights (airlines and airports by country)
  • Open-Meteo API (live weather by country centroid)
  • Public World GeoJSON (map geometry)
  • Google News RSS (country news)

Extended datasets (IMF, WHO, ND-GAIN, EM-DAT) are loaded as local snapshots to keep the UI fast. Visa/Residency path finder uses open regional mobility blocs plus country risk indicators as a pre-screen because no single free global visa policy API exists; always verify with official embassy/immigration portals. Relocation budget is an analytic estimate based on World Bank consumption/PPP proxies plus risk adjustments, not legal or financial advice. Military aircraft counts are not available from any free global public API, so this tool does not fabricate that metric.
